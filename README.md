# Descripción del repositorio

El material del repositorio consta de los códigos de apoyo para el curso de modelación y análisis de sistemas y control automático, impartido por el profesor Samuel Vergara en la UCSC (Concepción, Chile).

El objetivo es que los estudiantes puedan aprender a utilizar Python para resolver problemas de control, dado que es un lenguaje más transversal que matlab.
